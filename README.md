# excel-to-xml-to-data-Project
将excel转为可使用数据

前言：
现在基本都是用protobuf比较方便，但是现在还是有一些老的项目用xml，这里提供一种可以将excel直接转成可用数据的方法，不需要写一行代码。
excel转xml用的vb，up主没有写过博客所以如果有幸被转载请注明出处thanks: https://github.com/akukua/excel-to-xml-to-data-Project

使用环境:Excel 2007

使用：
excel按 Alt + F8使用宏
